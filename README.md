# Cell Senescence Detection & DNA Isolation Techniques

## 📌 Overview
This repository contains my internship work carried out at the **Basic Science Research Centre (BSRC), KLE Belagavi** as part of my Bachelor of Engineering in Biotechnology (VTU).

The work focuses on:
- DNA isolation from plant cells (spin column method)
- DNA isolation from animal cells (TRIzol method)
- Detection of cellular senescence using SA-β-galactosidase assay
- A review on exome-based immunotherapy against cancer

---

## 🧪 Techniques & Methods
- Plant DNA Isolation (Silica membrane-based kit)
- Animal DNA Isolation (TRIzol reagent)
- Nanophotometer-based DNA quantification
- Cell senescence detection using EZdetect™ SA-β-gal kit
